# Sistema de Gestão para Petshop

Este projeto consiste no desenvolvimento de um **sistema web de gestão para petshop**, criado como parte de uma atividade de **curricularização da extensão universitária**.

# 🐾 Contexto do Projeto

Este sistema está sendo desenvolvido gratuitamente para atender as necessidades do **petshop Fish Aqua Pets**, um estabelecimento real, como parte de uma atividade acadêmica de **curricularização**, que tem como objetivo aproximar o aprendizado universitário de demandas da comunidade.

O projeto busca oferecer uma solução tecnológica que contribua para melhorar a organização e gestão do negócio.

---

# 📌 Objetivo do Projeto

O objetivo é desenvolver um sistema que auxilie na **gestão operacional do petshop**, permitindo organizar informações importantes como:

* gerenciamento de estoque de produtos
* histórico de atendimentos
* controle de serviços realizados

Além do desenvolvimento do software, o projeto também busca aplicar **conceitos de Engenharia de Software**, incluindo:

* organização de código
* arquitetura de sistemas
* versionamento
* documentação e boas práticas de desenvolvimento

---

# 🏗 Arquitetura do Sistema

O sistema segue uma arquitetura **cliente-servidor**, com separação entre frontend e backend.

```id="n55nns"
Frontend (HTML, CSS, JavaScript)
        │
        ▼
Backend API (Java + Spring Boot)
        │
        ▼
Banco de Dados (PostgreSQL)
```

---

# 📁 Estrutura do Repositório

O projeto utiliza um modelo **monorepositório**, onde frontend e backend ficam no mesmo repositório.

```id="xjghaz"
gestao-estoque-petshop/
├── demais-arquivos/         #arquivos auxiliares gerados durante o desenvolvimento do projeto
├── documentacao/            #versionamento da documentação e diagramação de processos
├── petshop-system/          #sistema
│   └── backend/
│       └── petshop-api/
|   └── frontend/
├── .gitattributes
├── LICENSE
└── README.md
```

* **backend** → API desenvolvida em Java utilizando Spring Boot
* **frontend** → interface web do sistema
* **README.md** → documentação inicial do projeto
